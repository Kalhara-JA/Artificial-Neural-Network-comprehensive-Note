# 🧠 ANN Interactive Study Notes

> Personal interactive study notes I generated for my Artificial Neural Networks exam. Not a textbook — just everything I needed to actually understand the concepts, visualised and worked through properly.

## What's Inside

A single self-contained HTML file covering the full ANN syllabus from scratch:

- Introduction to AI & intelligent systems
- Types of learning — supervised, unsupervised, reinforcement, self-supervised
- Data preprocessing for neural networks
- Biological neuron → artificial neuron
- Performance evaluation — regression & classification metrics
- Training mechanics — batches, epochs, IID assumption
- Overfitting, underfitting & early stopping
- Activation functions — Sigmoid, Tanh, ReLU, SiLU, Softmax
- Single Layer Perceptron (SLP) with training rule
- Multi-Layer Perceptron (MLP) with matrix forward propagation
- Backpropagation — full chain rule walkthrough
- Loss functions & gradient descent optimizers
- Weight initialization — Xavier, He/Kaiming
- Recurrent Neural Networks — BPTT, LSTM, Transformers
- Evolutionary computing
- Exam quick reference & formula sheet

## How to Use

Just open the HTML file in any browser. No setup, no dependencies, no internet required.
```bash
git clone https://github.com/yourusername/ann-study-notes
# then open ANN_Complete_Study_Notes.html in your browser
```

## Features

- **SVG neural network diagrams** — proper visualisations of SLP, MLP, RNN architectures
- **Worked calculations** — forward pass, full backprop, confusion matrix, metrics all computed step by step with real numbers
- **Clickable sidebar navigation** — jumps between sections, highlights current position
- **Colour-coded callouts** — tips, warnings, and calculation boxes visually separated
- **Comparison tables** — activation functions, optimizers, encoding methods, batch types
